# Watcher of Friends Online

### About script

This code displays your vk friends who online. 

### Requirements

Python 3.5

vk==2.0.2

### Getting started

* Install module vk-2.0.2
      
         pip install vk

* Run the script in a terminal

         python lang_frequency.py

* Enter your vk login and password
